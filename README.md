# YouTube Auto-Liker

A Chrome Extension that automatically likes YouTube videos after watching 40% of the content.

## Installation

1. Download or clone this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable **Developer mode** (toggle in top-right corner)
4. Click **Load unpacked**
5. Select the folder containing this extension

## Usage

1. Navigate to any YouTube video
2. Watch the video normally
3. When you reach 40% watch progress, the video will be automatically liked
4. Check the browser console (F12) to see: `[YouTube Auto-Liker] Video liked`

## Notes

- This extension is for **personal, local use only**
- No data is stored or transmitted
- No background scripts or service workers
- Works only on `https://www.youtube.com/watch*` pages
