# YouTube Auto-Liker

A Chrome Extension that automatically likes YouTube videos after watching 90% of the content.

## Installation

1. Download or clone this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable **Developer mode** (toggle in top-right corner)
4. Click **Load unpacked**
5. Select the folder containing this extension

## Notes

- This extension is for **personal, local use only**
- No data transmitted only last 50 videos are stored in local storage
- No background scripts or service workers
- Works only on `https://www.youtube.com/watch*` pages
